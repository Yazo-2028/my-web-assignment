# Minoras Bakery Website

## Introduction
Minoras Bakery is a small and upcoming local bakery based in Johannesburg that offers a variety of high-quality baked goods including desserts, pastries, cakes, bread, and coffee.

## Purpose of the Website
The purpose of this website is to provide an online platform for customers to view products, place orders, and access important information such as location and contact details.

## Target Audience
The target audience includes local residents, students, working individuals, and families in Johannesburg.

## Features
- Homepage
- Menu page
- Online ordering system
- Contact page
- Gallery
- About Us page

## Design Ideas
The website will use soft colours such as brown and cream, with a clean and easy-to-navigate layout.
